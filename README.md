
# Enterprise Compliance AI Agents Platform

An enterprise-grade multi-agent AI system for fraud detection and regulatory compliance built using **Azure AI Foundry**, **Microsoft Agent Framework**, and **Databricks Lakehouse architecture**.

This platform demonstrates how AI agents can automate financial compliance workflows including transaction monitoring, risk scoring, regulatory reporting, and fraud alert generation.

---

# Azure AI Foundry Integration

This project uses **Azure AI Foundry models** to power the intelligent decision-making of the agents.

The deployed model:

Model: GPT-4.1-mini  
Platform: Azure AI Foundry  
Endpoint: Azure OpenAI Service

The model is used by agents to:

• analyze transaction patterns  
• detect suspicious activity  
• generate compliance explanations  
• produce regulatory reports
